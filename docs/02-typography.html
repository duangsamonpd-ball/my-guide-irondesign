Montserrat — Content & UI

One family for everything: nine content levels (H1 → Code) and six UI levels (buttons, nav). Ships the full 100–900 weight range, so every weight below is real — no fake-bold. Hierarchy is driven by weight and size together, stepping 900 / 800 / 700 / 600 so each level stays distinct.
