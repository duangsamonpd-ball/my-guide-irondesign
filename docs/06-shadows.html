🌑 Shadow

Elevation layers for cards, dropdowns, modals and focus rings. Use the right shadow level to communicate depth and hierarchy.

Focus Rings (Accessibility)

Focus rings appear when a user navigates with keyboard. Each button colour has a matching ring so it's always visible — this is a WCAG accessibility requirement.

Primary Button

focus-magenta

--shadow-focus-magenta
iron-pink-500 · Primary CTA
rgba(224, 26, 89, 0.25) · spread: 3px
Secondary Button

focus-blue

--shadow-focus-blue
iron-blue-500 · Secondary / Info
rgba(42, 149, 213, 0.25) · spread: 3px
Destructive

focus-red

--shadow-focus-red
iron-red-500 · Danger actions
rgba(229, 36, 42, 0.25) · spread: 3px
💡 Why do focus rings matter?

Users who can't use a mouse navigate websites with the keyboard. Focus rings show them which element is currently selected — like a cursor for keyboard users. Without them, the site becomes inaccessible.
