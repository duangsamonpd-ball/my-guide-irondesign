🎯 Semantic Colors

Colors assigned a purpose — not just a value. Maps each role (Primary, Text, Surface, Feedback) to its palette token so designers and developers always reach for the right color.

💡

Raw scale vs. semantic token

The Color Palette lists every raw shade (50–950). This page maps those shades to meaningful roles — Primary, Heading, Background, etc. — so designers and developers always reach for the right color for the right job.
