Tooltip

A short piece of contextual text that appears on hover or focus, next to an element. Four placements and an optional link — hover a trigger below to see it live.

Code
Copy-and-own — HTML markup, or the token-bound CSS. Show exactly one .tt-arrow per instance.
<span class="tt show-on-hover">
  <button class="tt-trigger">Hover me</button>
  <span class="tt-bubble above">
    <span class="tt-arrow arrow-bottom"></span>
    <!-- .tt-content holds the rounded, filled box; keeps the arrow un-clipped -->
    <span class="tt-content">
      <!-- omit .tt-title entirely for a compact, title-less tooltip -->
      <span class="tt-title">Title</span>
      <span class="tt-body">Supplementary detail here.</span>
      <!-- optional -->
      <a class="tt-link" href="#">Learn more →</a>
    </span>
  </span>
</span>
.tt-bubble {   /* positions the arrow + content; no fill of its own */
  position: absolute; width: var(--tt-w);   /* .max => var(--tt-w-max) */
}
.tt-content {   /* the visible rounded box — separate so overflow:hidden */
  background: var(--tt-bg);               /* doesn't clip the arrow poking out of .tt-bubble */
  border-radius: var(--tt-radius);
  box-shadow: var(--tt-shadow);
  overflow: hidden;                      /* clips title's square corners to match */
}
.tt-title {   /* darker header bar — neutral/950, own fill */
  padding: 8px 16px; font-size: 14px; font-weight: 700;
  color: var(--tt-title); background: var(--tt-title-bg);
}
.tt-body  { padding: 12px 16px 16px; font-size: 12px; color: var(--tt-body); }   /* no own fill */
.tt-link  { font-size: 12px; font-weight: 600; color: var(--tt-link); }
.tt-link:hover { color: var(--tt-link-hover); }   /* blue → pink, same as Button outline/ghost */
.tt-arrow.arrow-bottom {   /* tooltip above trigger; direct child of .tt-bubble, not .tt-content */
  border-top: 6px solid var(--tt-arrow);
  border-left: 6px solid transparent; border-right: 6px solid transparent;
}
Do & Don't
✓ Do
  • Use tooltips to surface supplementary, non-critical information.
  • Keep tooltip text concise — one to two short sentences maximum.
  • Position the tooltip close to its trigger element.
✗ Don't
  • Don't place critical or required information inside a tooltip — it may never be seen.
  • Don't use long paragraphs or multiple lines of dense text.
  • Don't rely on tooltips for mobile interactions where hover is unavailable.
