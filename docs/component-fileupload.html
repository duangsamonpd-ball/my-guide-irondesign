File Upload

A dashed dropzone for attaching a file — drag-and-drop or click to browse. Used in longer forms (quote requests, migration questionnaires) where a document helps the team respond faster.

Guidelines
✓ Do
✛
Drag & drop your file
  • Always show the accepted size / file type limit up front.
  • Support both drag-and-drop and click-to-browse — never drag-only.
  • Confirm the selected file (name + size) before submit.
✗ Don't
!
Upload failed
  • Don't fail silently — always show why an upload was rejected.
  • Don't make the dropzone the only way to attach a file.
  • Don't hard-code the tint colours; use the token opacity values.
