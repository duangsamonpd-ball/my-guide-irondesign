Radio

Lets the user pick exactly one option from a set. Same token language as Checkbox, but circular and mutually exclusive. The groups below are live — only one can be on at a time.

Guidelines
✓ Do
  • Use for one-of-many — exactly one selected at a time.
  • Always give the group a shared name.
  • List 2–6 options; beyond that, use a Select.
✗ Don't
  • Don't use a radio for an on/off setting — that's a toggle.
  • Don't allow multiple selections — that's a checkbox.
  • Don't hard-code the selected colour; use the token.
