The Shade Scale

What each number means and when to use it

50–100 lightest

Background & Tints

So light they're almost invisible. Use for card backgrounds, table stripes, info banners, hover states on white.

200–300 soft

Borders, Dividers & Tags

Visible but not aggressive. Perfect for borders, dividers, subtle badges, and decorative underlines.

400 mid-light

Secondary Actions & Icons

Lighter variant of the main color. Works for secondary buttons, icon fills, link underlines, and active states in dark mode.

500 pure

Primary Buttons & Links

The "true" color. This is your main CTA button, primary link, and brand highlight. Most recognizable shade of each color.

600–700 deep

Hover & Active States

Darker step of the primary. Use for hover: on buttons, pressed states, focused inputs, and sidebar nav items.

800–950 darkest

Dark Backgrounds & Body Text

Near-black versions of each hue. Use for dark mode surfaces, deep nav bars, and high-contrast body text on white.
