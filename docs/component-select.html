Select

A trigger that opens a menu to pick one value from a longer list — countries, plans, statuses. Click the trigger below to open it; the chevron flips and the chosen option is ticked.

Guidelines
✓ Do
  • Use when there are more than ~6 options.
  • Always show the current value in the trigger.
  • Keep option labels short and scannable.
✗ Don't
  • Don't use a Select for 2–3 options — use Radio.
  • Don't open the menu without a visible chevron cue.
  • Don't hard-code the menu shadow; use --shadow-float.
