Form Card

A composed wrapper for icon + title lead-capture forms — the "Request a Quote" / "Tell us what you're migrating from" pattern. Drop any Input, Select, Textarea or File Upload fields into the slot; the card supplies the frame, heading, submit and footnotes. The card below is live.

Guidelines
✓ Do

Start Free Trial

One clear ask per card.

  • Keep one primary action per card — a single submit.
  • Use the subtitle to set expectations, notes to reduce friction.
  • Match the icon to the intent: quote, key or check.
✗ Don't

Sign up · Log in · Learn more

Too many competing asks.

  • Don't stack multiple submits or CTAs in one card.
  • Don't hard-code the pink submit fill — use --color-primary.
  • Don't drop the visible labels on the slotted fields.
