Notice

A non-interactive informational block — disclaimers, notes, tips, callouts. Two shells for two contexts, five intents shared by both. Only the icon (and, on Bordered, the left bar) carries the intent colour — text always stays neutral.

Variants
Pick by context. Both share the same five intents and icon colours — only the shell and text placement differ.
Filled callout card
A tinted, rounded card. Icon + bold title sit on their own line, with the paragraph below — reads as a distinct block, good for standalone disclaimers and dedicated notes sections.

Guidelines
✓ Do
Tips
  • Match the intent colour to the meaning (info ≠ warning ≠ danger).
  • Use Bordered inline within long-form content, Filled as a standalone callout.
  • Keep the title/label neutral — only the icon (and bar) should carry colour.
  • Icons are Font Awesome Free Solid SVGs, inlined with fill="currentColor" — no icon font or CDN dependency.
✗ Don't
Click here for details →
  • Don't make a Notice clickable — that's a Button or a real link.
  • Don't colour the title/paragraph text — that breaks the icon-only colour convention.
  • Don't use danger/warning for neutral information.
