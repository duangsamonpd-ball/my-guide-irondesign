Notice

A non-interactive informational container — disclaimers, notes, short callouts. Three content patterns share the same tinted-card shell: title + body, inline lead-in + text, or a minimal label.

Guidelines
✓ Do
✓
Check
  • Match the intent colour to the meaning (info ≠ warning ≠ danger).
  • Use the minimal pattern for short, self-evident callouts.
  • Reach for title + body only when the disclaimer genuinely needs it.
✗ Don't
✕
Click here for details →
  • Don't make a Notice clickable — that's a Button or a real link.
  • Don't stack more than one Notice in the same view without a good reason.
  • Don't use danger/warning for neutral information.
