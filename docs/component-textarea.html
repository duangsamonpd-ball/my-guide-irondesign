Textarea

A multi-line text field for longer input — messages, descriptions, feedback. Shares the Input field's tokens (border, focus ring, error, disabled) so the two always feel like one family.

.textarea:disabled { background: #F8FAFC; color: var(--color-text-disabled); cursor: not-allowed; resize: none; }
Guidelines
✓ Do
  • Give it a visible label, same as Input.
  • Set a sensible default height for the expected content length.
  • Let people resize it vertically — don't disable resize without reason.
✗ Don't
  • Don't use a Textarea for single-line input — that's an Input.
  • Don't hard-code the focus colour; use the token.
  • Don't make it so short the placeholder gets clipped.
