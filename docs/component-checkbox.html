Checkbox

A control that lets the user toggle between checked and not checked — for accepting terms, multi-select lists and selectable cards. Try clicking any checkbox below; they're live.

Guidelines
✓ Do
  • Always pair the box with a clickable label.
  • Use for multi-select — many options can be on at once.
  • Show the red invalid border only after a failed submit.
✗ Don't
  • Don't use a checkbox for one-of-many — that's a radio.
  • Don't use it as an on/off switch — that's a toggle.
  • Don't leave the box without a label (bad a11y + tiny target).
