Text Link

An inline, underlined link for reference tables and code documentation — modelled after Tailwind's docs quick-reference table. Same underline treatment on light or dark backgrounds; only the token pairing changes.

Modes
Light-background links use --color-text-link; dark-background links (code blocks, dark sections) use --color-text-dark-link — currently the same value, kept as separate tokens for when dark mode diverges.

  text-decoration: underline;
  text-underline-offset: 3px;
  font-weight: 600;
  transition: color .15s;
}
.tlink:hover { color: var(--color-text-link-hover); }
.tlink:focus-visible { outline: 2px solid var(--color-text-link); outline-offset: 3px; border-radius: 2px; }
.tlink.dark { color: var(--color-text-dark-link); }
.tlink.dark:hover { color: var(--color-text-dark-link-hover); }
Guidelines
✓ Do
focus-visible
  • Always underline — colour alone isn't enough contrast for a link.
  • Use .dark on dark backgrounds (code blocks, dark sections) so contrast holds.
  • Add the external-link icon (.ext-icon) when the link leaves the current site.
✗ Don't
focus-visible
  • Don't drop the underline on a plain-text link — it stops reading as a link.
  • Don't use the light-mode token on a dark background — contrast fails.
  • Don't use this for buttons — that's a Button component.
